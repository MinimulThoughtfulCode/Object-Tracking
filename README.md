# Object-Tracking
Python Script that Tracks Objects in Real Time

# How to Use

Procedures
1. Find and download your own video clip
2. Run this program in your terminal
4. Click 's' and drag with your crosshair to track
5. Click the SPACE BAR to continue
6. Repeat steps 4-5 track multiple objects

Terminal Prompt Example:

python3 object_tracking.py --video clip.mp4
